The Narnoo distributor plugin gives a tourism organisation and tourism wholesalers the ability to connect to their products, promotional media hosted on Narnoo.com.

== Description ==

The Narnoo distributor plugin allows a tourism organisation, wholesaler, online booking site to connect with their product's promotional media hosted on Narnoo.com. Using the plugin Distributor's can manage their operator lists, access their operator's media in real time and display this media throughout their WordPress pages and blogs. The plugin allows the distributor to manage their own promotional media within Narnoo as well. Narnoo provides shortcodes that make displaying image galleries, videos and brochures simple and easy to achieve. All media files are hosted on Narnoo's global servers therefore, freeing up the users server requirements.



== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
2. Retrieve your APP authentication keys from your account at Narnoo.com. You will need your Access Key and corresponding Secret Key. Log in to your Narnoo.com account -> My Account -> either create a New APP or View Applications.
3. Enter your APP keys in Settings -> Narnoo API -> API  Settings area.
4. Your WordPress site is now connected to your Narnoo.com account.  

== Frequently asked questions ==

How to I get my Narnoo APP keys?

You can retrieve your APP keys from your account on Narnoo.com. Simply log into your account and go to the My Account -> View Applications menu option. From here you can either create or view existing key pairs.
